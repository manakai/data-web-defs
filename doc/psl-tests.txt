data/psl-tests.json
~~~~~~~~~~~~~~~~~~~

Test data for the operation on Public Suffix List.

* Data structure

The file contains a JSON array of arrays, which contains the input and
the expected output, which are strings or |null| values.

* Sources

View the Public Suffix List <https://publicsuffix.org/list/>.

<https://raw.githubusercontent.com/publicsuffix/list/master/tests/test_psl.txt>.
