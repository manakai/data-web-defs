data/selectors.json
~~~~~~~~~~~~~~~~~~~

* Structure

The file contains a JSON object with following name/value pair:

  attr_value_case_insensitive

    The list of the attributes which is evaluated ASCII
    case-insensitively when used in an attribute selector matched with
    an HTML element in an HTML document.  Names are attribute names
    and values are [boolean] true values.

* Source

HTML Standard <https://html.spec.whatwg.org/>.
