data/tls-certs.pem
~~~~~~~~~~~~~~~~~~

This file is a collection of PKIX root CA certificates, which is
intended for TLS client applications of the Web.

This is a snapshot copy of Mozilla's CA certificates.

* Sources

Mozilla::CA Perl module <https://github.com/gisle/mozilla-ca>.

CA/Included Certificates - MozillaWiki,
<https://wiki.mozilla.org/CA/Included_Certificates>.
