dom-perl.json
~~~~~~~~~~~~~

This file contains data related to the manakai's DOM Perl Binding
specification.

* Structure

The file contains following name/value pair:

  method_name_map [object]

    Names are method or attribute names in WebIDL, while values are
    corresponding method names in Perl.

    This data is normative part of the DOMPERL spec.

* Reference

[DOMPERL] manakai's DOM Perl Binding
<http://suika.suikawiki.org/~wakaba/wiki/sw/n/manakai's%20DOM%20Perl%20Binding>.
