data/temma-syntax.txt
~~~~~~~~~~~~~~~~~~~~~

The file contains data on the Temma syntax.  See |elements.json| for
Temma elements and attributes.

* Data structure

The file contains a JSON object, which contains following name/value
pair:

  tokenizer [object]

    A machine-readable description of the Temma tokenizer.  See
    |doc/html-syntax.txt|'s description for |tokenizer|.
