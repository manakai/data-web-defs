xhtml-charrefs.dtd
~~~~~~~~~~~~~~~~~~

This file is the XML DTD containing the HTML named character
references, as used by XML parsers.

See also |data/xml-syntax.json|, which contains the public identifiers
for the DTD.

* Source

HTML Standard
<http://www.whatwg.org/specs/web-apps/current-work/#parsing-xhtml-documents>.
