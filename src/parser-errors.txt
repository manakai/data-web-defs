* stray end tag

name=after-body-end-html
value=tag_name

@en
The tag <code>&lt;/<var>{value}</var>></code> has no corresponding
start tag
@ja
終了タグ <code>&lt;/<var>{value}</var>></code>
に対応する開始タグがありません

@en

<p>The end tag has no corresponding start tag.  There must be a start
tag with the same tag name before the end tag.  The end tag is
ignored.</p>

<p>Note that void elements have no end tag, that is, following
elements cannot have end tags: <code>area</code>, <code>base</code>,
<code>br</code>, <code>col</code>, <code>embed</code>,
<code>hr</code>, <code>img</code>, <code>input</code>,
<code>keygen</code>, <code>link</code>, <code>menuitem</code>,
<code>meta</code>, <code>param</code>, <code>source</code>,
<code>track</code>, and <code>wbr</code>.</p>

@ja

<p>この終了タグには対応する開始タグがありません。
終了タグより前に同じタグ名の開始タグが必要です。終了タグは無視されます。</p>

<p>なお void 要素は終了タグを持ちませんから、次の要素では終了タグを使えません:
<code>area</code>, <code>base</code>, <code>br</code>,
<code>col</code>, <code>embed</code>, <code>hr</code>,
<code>img</code>, <code>input</code>, <code>keygen</code>,
<code>link</code>, <code>menuitem</code>, <code>meta</code>,
<code>param</code>, <code>source</code>, <code>track</code>,
<code>wbr</code>。</p>

* doctype:bad context

name=after-body-doctype
name=after-frameset-doctype

@en
The <code>DOCTYPE</code> is not at the beginning of the document
@ja
<code>DOCTYPE</code> が文書の先頭ではありません

@en

<p>There must not be anything other than white space characters and
comments before the <code>DOCTYPE</code>.  It cannot appear after
another <code>DOCTYPE</code> or within an element.

@ja

<p><code>DOCTYPE</code> の前に置けるのは空白と注釈だけです。
<code>DOCTYPE</code> が2つあったり、要素の中にあったりしてはいけません。

name=after-after-body-else
name=after-after-frameset-else

@en

@ja
<code>&lt;/html></code> 終了タグの後にテキストやタグがあります

@en

<p>Nothing other than white space characters and comments are allowed

@ja

<p><code>html</code> 要素の終了タグの後には、空白と注釈以外は何も書けません。

* after-body-else

name=after-body-else
name=after-frameset-else

@en
There is a text or tag after the end tag of the body element
@ja
文書 <code>body</code> 要素の終了タグの後にテキストやタグがあります

@en

<p>Nothing other than the end tag of the <code>html</code> element,
white space characters, and comments are allowed after the end tag of
the <code>body</code> element or the outermost <code>frameset</code>
element.

@ja

<p><code>body</code> 要素や最外 <code>frameset</code> 要素の終了タグの後には、
<code>html</code> 要素の終了タグと空白、注釈以外は何も書けません。
