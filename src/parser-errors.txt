* stray end tag

name=after-body-end-html
value=tag_name

@en
The tag <code>&lt;/<var>{value}</var>></code> has no corresponding
start tag
@ja
終了タグ <code>&lt;/<var>{value}</var>></code>
に対応する開始タグがありません

@en

<p>The end tag has no corresponding start tag.  There must be a start
tag with the same tag name before the end tag.  The end tag is
ignored.</p>

<p>Note that void elements have no end tag, that is, following
elements cannot have end tags: <code>area</code>, <code>base</code>,
<code>br</code>, <code>col</code>, <code>embed</code>,
<code>hr</code>, <code>img</code>, <code>input</code>,
<code>keygen</code>, <code>link</code>, <code>menuitem</code>,
<code>meta</code>, <code>param</code>, <code>source</code>,
<code>track</code>, and <code>wbr</code>.</p>

@ja

<p>この終了タグには対応する開始タグがありません。
終了タグより前に同じタグ名の開始タグが必要です。終了タグは無視されます。</p>

<p>なお void 要素は終了タグを持ちませんから、次の要素では終了タグを使えません:
<code>area</code>, <code>base</code>, <code>br</code>,
<code>col</code>, <code>embed</code>, <code>hr</code>,
<code>img</code>, <code>input</code>, <code>keygen</code>,
<code>link</code>, <code>menuitem</code>, <code>meta</code>,
<code>param</code>, <code>source</code>, <code>track</code>,
<code>wbr</code>。</p>

always be used.  Use of <code>PUBLIC</code> or <code>SYSTEM</code>

<p>DOCTYPE は <code>&lt;!DOCTYPE html></code> とだけあれば十分です。00

!!!
